## 2. 모바일 UX 심화 검토

### ✅ 잘 된 점
- 엄지 영역 내 핵심 조작 배치
- 터치 타겟 48×48px 기준 설정
- 2~3분 세션 길이 설계
- 3탭 이내 코어 루프 경험 온보딩
- 원핸드 조작 가능 설계

### ⚠️ 추가 검토 필요

| 항목 | 권고 |
|---|---|
| **가로/세로 모드** | 러너 게임 특성상 가로 고정 권장. 세로 모드는 가로 전환 유도 UI 필요 |
| **노치/펀치홀** | Safe Area Inset 적용하여 HUD가 노치에 가리지 않도록 |
| **저사양 기기 폴백** | WebGL 미지원 시 Canvas2D 자동 폴백 (Phaser 내장) |
| **배터리/발열** | 60fps 고정이 아닌, 기기 성능에 따른 적응형 프레임레이트 검토 |
